Resources & Glossary

Glossary

Pointer
A variable that stores the memory address of another variable.
Dereference
Accessing the value stored at the memory address a pointer refers to.
Heap
Memory used for dynamic allocation via malloc/calloc/realloc.
Stack
Memory used for function calls, local variables, and return addresses.
Segmentation Fault
A crash caused by accessing invalid memory (e.g., null pointer dereference).
