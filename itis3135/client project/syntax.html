Basic Syntax

Before diving into pointers and memory, you need to understand the basic structure of a C program. C is small, fast, and minimal — which means you write more code yourself, but you also gain more control.

Program Structure

Every C program starts with a main() function. This is where execution begins. 


#include <stdio.h>

int main() {
    printf("Hello, world!\\n");
    return 0;
}

Variables & Types

C is statically typed, meaning every variable has a type known at compile time. 


int age = 20;
float gpa = 3.7;
char letter = 'A';

Control Flow

C includes familiar control structures like if, while, and for. 


for (int i = 0; i < 5; i++) {
    printf("i = %d\\n", i);
}

These basics form the foundation for everything else in C.
