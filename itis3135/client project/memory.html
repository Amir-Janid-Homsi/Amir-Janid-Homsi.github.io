Memory Allocation

C gives you manual control over memory using malloc, calloc, realloc, and free. This is powerful — and dangerous if misused — but it allows you to build flexible data structures.

malloc()

malloc allocates a block of memory and returns a pointer to it. 


int *arr = malloc(5 * sizeof(int));
if (!arr) {
    printf("Allocation failed!\\n");
    return 1;
}

Using the Memory


for (int i = 0; i < 5; i++) {
    arr[i] = i * 10;
}

Freeing Memory


free(arr);

Common Mistakes

  • Forgetting to free memory → memory leak
  • Freeing memory twice → undefined behavior
  • Using memory after freeing it → use-after-free bug
  • Allocating the wrong size → buffer overflow

Mastering dynamic memory is essential for real-world C programming.
